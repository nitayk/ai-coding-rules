# Code Optimization

Optimize code for performance, memory usage, or efficiency.

## When to Use

**APPLY WHEN:** User explicitly asks for performance optimization, memory efficiency, or algorithmic improvement.

**SKIP WHEN:** General cleanup (use `/simplify` or `/code-cleanup`), structural refactoring (use `/service-refactoring`).

## Process

1. Profile or measure current behavior
2. Identify bottlenecks
3. Apply targeted optimizations
4. Verify improvement with measurements
